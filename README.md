vis-bonjour-network
==========================
Visualize your local network using bonjour protocol

![screenshot](./screenshot.png)

**Run the Node.JS server**

```sh
git clone git@github.com:wyvernnot/vis-bonjour-network.git
cd vis-bonjour-network
npm install
npm start
```

**Open browser**

```
http://127.0.0.1:3000
```
